Конвертер величин - Задача для изучения JS/HTML/CSS
***
Интейфейс:
 - Выпадающий список с группами величин, есть элемент по умолчанию (Температура*, Длина, Масса, ...)
 - 2 выпадающих списка с единицами измерения, нельзя выбрать одинаковые, в первом списке можно выбрать любую ЕИ, есть элементы по умолчанию
 - Поле ввода конвертируемого значения, можно ввести только цифры/запятую/минус, по умолчанию "1"
 - Label для вывода результата конвертации, значение пересчитывается при изменении значения в поле ввода/в выпадающих списках
 - Кнопка "поменять местами ЕИ", не изменяет значение в поле ввода конвертируемой величины
